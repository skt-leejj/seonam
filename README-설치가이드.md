# 서남센터 축구장 예약 도우미 v2.2

서울시 공공서비스예약 사이트(yeyak.seoul.go.kr)에서 서남센터 축구장 예약을 단계별로 보조하는 Tampermonkey 스크립트입니다.

---

## 설치

### 1. Tampermonkey 확장 프로그램 설치 (최초 1회)

| 브라우저 | 설치 경로 |
|---------|----------|
| Chrome | Chrome 웹스토어 → "Tampermonkey" 검색 → 설치 |
| Edge | Edge 추가 기능 → "Tampermonkey" 검색 → 설치 |
| Firefox | Firefox 부가 기능 → "Tampermonkey" 검색 → 설치 |

설치 후 브라우저 우측 상단에 검은 사각형 아이콘이 나타납니다.

### 2. 스크립트 등록

**방법 A — 파일 드래그 (추천)**
1. `seonam-reservation-helper.user.js` 파일을 브라우저 창에 드래그 앤 드롭
2. Tampermonkey 설치 화면에서 **"설치"** 클릭

**방법 B — 코드 직접 입력**
1. Tampermonkey 아이콘 → "새 스크립트 만들기"
2. 기존 내용 모두 삭제 → 스크립트 코드 붙여넣기 → `Ctrl+S` 저장

---

## 사용 방법

### 사전 준비
- 서울시 공공서비스예약 사이트에 **로그인**
- 서남센터 축구장 페이지 접속:
  `https://yeyak.seoul.go.kr/web/reservation/selectReservView.do?rsv_svc_id=S210407133421401399`

### 전체 흐름

```
 1페이지 (시설 상세)                2페이지 (예약 신청)
┌────────────────────┐           ┌────────────────────────────┐
│                    │           │ [1] 날짜 선택         직접  │
│ 날짜/시간/인원 설정  │           │ [2] 회차 선택         직접  │
│                    │           │ [3] 인원 선택         직접  │
│ 날짜 선택 가능 여부  │           │ [4] 동의체크+CAPTCHA포커스  │
│ 자동 체크           │           │     [4.동의+포커스] 클릭    │
│                    │           │ [5] CAPTCHA 입력      직접  │
│ 하단 [새로고침]     │ ───────→ │ [6] [6.예약!] 클릭         │
│ 하단 [실행] 클릭    │           │                            │
│                    │           │ * 버튼은 패널 하단에 크게    │
│ 미오픈시 자동새로고침 │           │   배치되어 클릭이 편리합니다 │
└────────────────────┘           └────────────────────────────┘
```

### 1페이지: 시설 상세

화면 우측 상단에 패널이 표시됩니다.

1. **날짜** — 이용하고 싶은 날짜 선택
2. **시간** — 08:00~10:00 / 10:00~12:00 / 13:00~15:00 / 15:00~17:00
3. **인원** — 1~24명

날짜 입력 시 자동으로 선택 가능 여부를 체크합니다:
- **선택 가능** (오픈됨) → 상태 표시: `✔ 날짜 선택 가능!` + 하단 **[실행]** 버튼 활성화 (초록색)
- **선택 불가** (미오픈) → 상태 표시: `✖ 해당 날짜 선택 불가 (미오픈)` + **[실행]** 버튼 비활성화 (회색)
- 캘린더가 비동기로 로딩되는 경우에도 DOM 변경을 자동 감지하여 재체크합니다

패널 하단에 두 개의 큰 버튼이 있습니다:
- **[새로고침]** — 페이지를 새로고침하여 캘린더 상태를 다시 확인
- **[실행]** — 날짜 클릭 + 예약하기 클릭 → 2페이지로 자동 이동

실행 시:
- 날짜가 오픈되어 있으면 → 자동으로 날짜 클릭 + 예약하기 클릭 → 2페이지로 이동
- 아직 미오픈이면 → 3~5초 간격으로 자동 새로고침하면서 대기

> 설정값은 입력 시 자동 저장되어 새로고침 후에도 유지됩니다.

### 2페이지: 예약 신청

패널에 6단계가 표시되고, 하단에 큰 액션 버튼이 배치됩니다.

| 단계 | 내용 | 수행 방식 |
|:---:|------|:--------:|
| 1 | 달력에서 날짜 선택 | 직접 클릭 |
| 2 | 회차(시간대) 선택 | 직접 클릭 |
| 3 | 인원수 + 버튼 클릭 | 직접 클릭 |
| 4 | 동의 체크 + CAPTCHA 포커스 | **[4.동의+포커스] 버튼** |
| 5 | CAPTCHA 이미지 문자 입력 | 직접 입력 |
| 6 | 예약 제출 | **[6.예약!] 버튼** |

**순서:**
1. 1~3단계를 직접 수행
2. 하단 **[4.동의+포커스]** 클릭 → 동의 체크박스 자동 체크 + CAPTCHA 입력란 포커스 + 알림음
3. CAPTCHA 이미지 문자를 직접 입력
4. 하단 **[6.예약!]** 클릭 → 예약 제출

## 문제 해결

**패널이 안 보여요**
- Tampermonkey 아이콘 클릭 → 스크립트가 활성화 상태인지 확인
- 서남센터 축구장 페이지(yeyak.seoul.go.kr)에서만 동작합니다
- v2.1 미만 버전에서는 2페이지 진입 시 패널이 간헐적으로 안 나타날 수 있습니다 → 최신 버전으로 업데이트

**1페이지에서 실행 버튼이 비활성화(회색)예요**
- 입력한 날짜가 캘린더에 아직 오픈되지 않은 상태입니다
- [새로고침] 버튼을 눌러 캘린더를 다시 불러오세요
- 날짜가 오픈되면 자동으로 버튼이 활성화(초록색)됩니다

**4단계 실행 후 체크박스가 체크 안 된 것처럼 보여요**
- `.checked = true` 방식은 시각적으로 체크 표시가 안 보일 수 있으나 내부적으로 체크된 상태입니다
- 걱정되면 직접 클릭으로 체크해도 됩니다

**6단계에서 fn_Resev 함수 미발견이라고 나와요**
- CAPTCHA를 먼저 입력했는지 확인
- 페이지를 새로고침 후 1단계부터 다시 시도

**다른 사람에게 전달하려면?**
- `seonam-reservation-helper.user.js` 파일을 전달하면 됩니다
- 받는 사람도 Tampermonkey 설치 후 파일 드래그 앤 드롭으로 설치
